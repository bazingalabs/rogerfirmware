Roger v3 by bazingalabs

Bazingalabs Roger wireless module

The ROGERv3 is a radio module designed provide you with an easy way of adding wireless functionality to any device in a user friendly manner. All you need to be able to work with the ROGERv3 is a PC or MAC and our Bazingalabs PROGER module. We’ve created libraries for this module to make your life easier. All of the libraries are open source.

The newest firmware as well as the documentation can be found here

Key features

  • Available in 433 MHz, 868MHz and 915MHz radio frequency operation,
  • The module has the same footprint as Digi’s popular XBEE modules,
  • The module can be used as standalone device, because it incorporates a microcontroller (an ATMEGA328p),
  • Easy configuration,
  • Easy to use, even for people without any programming skills: the module can be operated by typing simple commands,
  • OpenSource, see: http://creativecommons.org/licenses/by/3.0/
  • Low power,
  • Bazinga ‘STACKY’ compatible: the ROGERv3 module can be combined with other Bazingalabs modules like motor controllers, processors, sensor boards, battery boards etc Just stack one module on top of one or more other modules and you’re ready to go!

Getting started guide

  1. Connect your Roger module to a arduino or other microcontroller as shown below
  2. Open the Arduino IDE
  3. Create a simple sketch like the example below
  4. Upload the example to your microcontroller
  5. Connect the other Roger module to a Proger or XBee Explorer
  6. Select the right port in the Tools->Serial Port menu
  7. Open the serial monitor
  8. Select the correct baudrate (57600 by default)
  9. You should now see the text “Hello World” appearing
void setup() {
  Serial.begin(57600);
}

void loop() {
  Serial.println("Hellow World");
  delay(1000);
}
